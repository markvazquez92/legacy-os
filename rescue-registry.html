Rescue Registry

Every dog that comes through this network gets a permanent record.

When the Lake Hughes fires hit, animals were displaced with no coordination system. We built this so it never happens again. Every rescue dog in our network gets a digital profile — photos, personality, medical needs, transport status, and adoption application — all in one scan.
